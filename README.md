## ✨ **المزايا الجديدة في البرنامج (سكربت Bash)**

### 🔔 **نظام إشعارات محسّن بالكامل**
- **إشعارات أذكار تلقائية** على فترات قابلة للتخصيص
- **تنبيهات قبل الصلاة** بـ 10-15 دقيقة مع صوت تنبيه
- **أذان كامل** عند دخول وقت الصلاة مع واجهة تحكم
- **سجل تفصيلي** لكل الإشعارات والصلاة

### 🕒 **حساب مواقيت الصلاة المتقدم**
- **دعم 22 طريقة حساب** مختلفة من جميع أنحاء العالم الإسلامي
- **اكتشاف تلقائي للموقع** عبر IP الجغرافي
- **تحديث يومي تلقائي** لمواقيت الصلاة
- **دقة عالية** في الحسابات مع مراعاة الفروق المحلية

### 🎵 **نظام صوتي متكامل**
- **مشغل أذان ذكي** مع دعم multiple players (mpv, ffplay, paplay, ogg123)
- **خيارات أذان متعددة** (كامل - قصير - بدون صوت)
- **أصوات تنبيه** لاقتراب وقت الصلاة
- **تكامل مع واجهات رسومية** للتحكم في التشغيل

### 🖥️ **تكامل مع جميع بيئات سطح المكتب**
- **دعم GNOME, KDE, XFCE** بالكامل
- **تكامل مع Window Managers** (i3, Sway, Awesome, bspwm, Openbox)
- **إشعارات نظام متكاملة** مع libnotify
- **واجهات رسومية** (zenity/yad/kdialog)

## 🛠 **أوامر البرنامج الجديدة والمحسنة**

### ⚙️ **إدارة متقدمة للبرنامج**
```bash
gtsalat --status          # 📊 عرض حالة متقدمة مع معلومات النظام
gtsalat --debug           # 🔍 معلومات تشخيص كاملة للنظام
gtsalat --settings        # ⚙️ إعدادات الموقع وطريقة الحساب
gtsalat --logs            # 📝 عرض سجل التشغيل المفصل
```

### 🎵 **نظام اختبارات شامل**
```bash
gtsalat --test-adhan      # 🔊 اختبار الأذان مع الواجهة الرسومية
gtsalat --test-approaching # ⏰ اختبار تنبيه اقتراب الصلاة
gtsalat --test-notify     # 🔔 اختبار إشعارات النظام
```

### 🔄 **تحديثات ذكية**
```bash
gtsalat --self-update     # 📥 تحديث تلقائي للبرنامج
gtsalat --update-azkar    # 📖 تحديث قائمة الأذكار من الإنترنت
```

### 📊 **عرض المعلومات**
```bash
gtsalat --show-timetable  # 🕒 عرض مواقيت الصلاة الكاملة
gtsalat                   # 🌟 عرض ذكر عشوائي + الصلاة القادمة
```
