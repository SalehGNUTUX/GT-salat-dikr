## 🚀 **أوامر التحكم الجديدة والمحسنة**

### 🔧 **أوامر التحكم الدقيق في الإشعارات**

#### 🧩 **أوامر عامة للإشعارات**
```bash
gtsalat --enable-all-notify       # تفعيل جميع الإشعارات (طرفية + نظام)
gtsalat --disable-all-notify      # تعطيل جميع الإشعارات
gtsalat --enable-salat-notify     # تفعيل إشعارات الصلاة فقط
gtsalat --disable-salat-notify    # تعطيل إشعارات الصلاة فقط
gtsalat --enable-zikr-notify      # تفعيل إشعارات الأذكار فقط
gtsalat --disable-zikr-notify     # تعطيل إشعارات الأذكار فقط
```

#### 💻 **إشعارات الطرفية (Terminal)**
```bash
gtsalat --enable-salat-terminal   # تفعيل إشعارات الصلاة في الطرفية
gtsalat --disable-salat-terminal  # تعطيل إشعارات الصلاة في الطرفية
gtsalat --enable-zikr-terminal    # تفعيل إشعارات الأذكار في الطرفية
gtsalat --disable-zikr-terminal   # تعطيل إشعارات الأذكار في الطرفية
```

#### 🪟 **إشعارات النظام (GUI)**
```bash
gtsalat --enable-salat-gui        # تفعيل إشعارات الصلاة في النظام
gtsalat --disable-salat-gui       # تعطيل إشعارات الصلاة في النظام
gtsalat --enable-zikr-gui         # تفعيل إشعارات الأذكار في النظام
gtsalat --disable-zikr-gui        # تعطيل إشعارات الأذكار في النظام
```

### ⚙️ **أوامر النظام المحسنة**
```bash
gtsalat --change-notify-system    # تغيير نظام الخدمة (systemd/sysvinit)
gtsalat --status                  # عرض حالة متقدمة مع كشف systemd
gtsalat --notify-start           # بدء الإشعارات حسب النظام المختار
gtsalat --notify-stop            # إيقاف الإشعارات حسب النظام المختار
```
